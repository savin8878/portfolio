# 🚀 Quick Start Guide - Premium Resume Builder

## Getting Started in 5 Minutes

### Step 1: Access the Resume Builder
1. Navigate to `/admin/resume-builder`
2. You'll see the main builder interface with:
   - Form editor on the left
   - Live preview on the right
   - Template gallery at the top

### Step 2: Choose Your Template
1. Click **"Choose Template"** button at the top
2. Browse 40+ professional templates
3. Filter by category: Original, Creative, Tech, Professional, etc.
4. Click a template to preview
5. Click **"Apply Template"** to use it

### Step 3: Fill in Your Information

#### Profile Section
- **Full Name** - Your complete name
- **Professional Title** - e.g., "Senior Software Engineer"
- **Email** - Professional email address
- **Phone** - Contact number
- **Location** - City, State/Country
- **Website** - Portfolio or personal website
- **LinkedIn** - LinkedIn profile URL
- **GitHub** - GitHub profile URL (for tech roles)
- **Professional Summary** - 2-3 sentences about yourself

#### Work Experience
1. Click **"Add Experience"**
2. Fill in:
   - Company name
   - Job title
   - Location
   - Start date / End date (or mark "Currently working here")
   - Job description
   - Achievements (add multiple)
3. Toggle **"Enabled"** to show/hide
4. Drag to reorder

#### Education
1. Click **"Add Education"**
2. Fill in:
   - Institution name
   - Degree
   - Field of study
   - Start and end dates
   - Achievements (GPA, honors, etc.)

#### Skills
1. Click **"Add Skill Category"** (e.g., "Languages", "Technical")
2. Add skills to each category
3. Reorder categories by dragging

#### Projects
1. Click **"Add Project"**
2. Fill in:
   - Project name
   - Description
   - Project URL
   - Technologies used

#### Certifications
1. Click **"Add Certification"**
2. Fill in:
   - Certification name
   - Issuing organization
   - Date earned

### Step 4: Customize Your Resume

#### Change Colors
1. Click the **color picker** next to accent color
2. Select your preferred color
3. See live preview update

#### Adjust Font Size
1. Use the **"Font Size"** dropdown
2. Options: Small, Normal, Large
3. Preview updates in real-time

#### Toggle Profile Image
1. Click checkbox to add profile image
2. Upload an image (optional)
3. Appears in relevant templates

### Step 5: Export Your Resume

#### Export as PDF
1. Click **"Download PDF"** button
2. Select filename
3. Click **"Download"**
4. Opens in your default PDF viewer
5. Print or share directly

#### Export as Word (.DOCX)
1. Click **"Download Word"** button
2. Select filename
3. Click **"Download"**
4. Opens in Microsoft Word
5. Edit further if needed

## 💡 Pro Tips

### Template Selection
- **Applying for a tech role?** → Try "Code Stack" or "Hacker"
- **Finance/Accounting?** → Try "Finance" or "Corporate Blue"
- **Healthcare?** → Use "Medical Pro" or "Healthcare"
- **Creative field?** → Try "Artist", "Aurora", or "Vibrant"
- **Executive position?** → Use "Executive Pro" or "Investor Ready"
- **Starting out?** → Use "Minimal" or "Clean" templates

### Content Tips
1. **Keep it concise** - Use bullet points, not paragraphs
2. **Use action verbs** - "Developed", "Led", "Designed"
3. **Include metrics** - "Increased sales by 25%"
4. **Tailor for each job** - Update skills section for job requirements
5. **Match keywords** - Use terms from job description
6. **Be honest** - Don't exaggerate experience

### Design Tips
1. **Stick to 2-3 colors maximum**
2. **Use readable fonts** - Avoid fancy fonts
3. **Consistent spacing** - Use templates' built-in spacing
4. **Don't overcrowd** - Leave white space
5. **Keep it professional** - Save creative designs for portfolios

### Export Tips
1. **PDF for emailing** - Everyone can open PDFs
2. **DOCX for ATS systems** - Better for parsing
3. **Multiple formats** - Export both PDF and DOCX
4. **Test download** - Verify file looks correct
5. **Use proper filename** - "FirstName_LastName_Resume.pdf"

## ❓ Frequently Asked Questions

### Q: Can I switch templates after filling in data?
**A:** Yes! Your data stays intact when switching templates. Just select a new template and your information appears in the new design.

### Q: Is my data saved?
**A:** Yes! Your resume data is saved to browser local storage automatically. It persists between sessions.

### Q: Can I download my data?
**A:** Yes! You can export your resume as JSON or import from a previous export.

### Q: What format is best for job applications?
**A:**
- PDF for most applications
- DOCX for applications asking for Word files
- Text only for systems that can't read formatted text

### Q: My PDF looks different than preview. Why?
**A:** This is usually due to:
- Font loading delays
- Different print settings
- Browser rendering differences
**Solution:** Clear cache and reload, or try a different template.

### Q: Can I edit the PDF after downloading?
**A:** PDFs are read-only. Edit in the builder instead, then re-export.

### Q: How many sections can I add?
**A:** Unlimited! Add as many jobs, skills, projects as you need. The system handles multi-page resumes automatically.

### Q: Is my data private?
**A:** Yes! All data stays in your browser. We don't collect or store your resume data.

## 🆘 Troubleshooting

### PDF Download Not Working
1. Check browser console (F12) for errors
2. Try a different browser
3. Clear browser cache
4. Try a simpler template

### Preview Not Updating
1. Click the refresh button
2. Clear local storage
3. Reload the page
4. Try a different template

### Colors Not Showing
1. Verify accent color is valid hex (#RRGGBB)
2. Try a template with color support
3. Check browser print settings
4. Clear cache and reload

### Text Appearing Cut Off
1. Try "Small" font size
2. Use a different template layout
3. Reduce summary/description length
4. Add content to multiple pages instead

### Fonts Look Wrong
1. Clear browser cache
2. Try refreshing page
3. Use system fonts in preferences
4. Check PDF viewer zoom level
